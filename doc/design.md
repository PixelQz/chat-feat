# 设计

## 合成系统
使用简单的合成公式进行合成
```rust
c = a x 10 + b x 2;
base on W in Zone of user.
```
公式包括一系列需要的staff(物品) 和需要的 特殊依赖。（工具）
在没有领地的情况下 要让物体在自己的多少个格子内！
如果在领地下。则从领地中查询。

UI！
合成是一个下来列表。有一个层级。
1. 全部公式
2. 角色解锁了的公式
3. 某个合成台生成后的公式
4. 可以合成的公式

检查公式是否可以合成。
1. 没有领地内的情况检查 toolbar和背包
2. 存在领地的情况下 在领地中的储物箱中获取。toolbar和背包
